# Клуб любителей творчества «ОчУмелые ручки»

Laravel-приложение для управления мастер-классами и записями пользователей.

## Установка

### Требования
- PHP 8.3+
- Composer
- SQLite (или другая СУБД)

### Шаги установки

1. **Клонировать или скопировать проект**
   ```bash
   cd /path/to/your/projects
   # Скопируйте папку с проектом
   ```

2. **Установить зависимости**
   ```bash
   composer install
   ```

3. **Настроить окружение**
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Настроить почту (опционально)**
   Отредактируйте `.env` файл для настройки почты:
   ```
   MAIL_MAILER=smtp
   MAIL_SCHEME=smtps
   MAIL_HOST=smtp.gmail.com
   MAIL_PORT=465
   MAIL_USERNAME=your-email@gmail.com
   MAIL_PASSWORD=your-app-password
   MAIL_ENCRYPTION=ssl
   MAIL_FROM_ADDRESS="your-email@gmail.com"
   MAIL_FROM_NAME="ОчУмелые ручки"
   MAIL_TIMEOUT=8
   SHOW_VERIFY_LINK_ON_REGISTER=true
   ```

5. **Запустить миграции и seeder**
   ```bash
   php artisan migrate --seed
   ```

   Это создаст таблицы БД, категории и двух инструкторов:
   - `instructor1@test.com` / `password123`
   - `instructor2@test.com` / `password123`

6. **Запустить сервер**
   ```bash
   php artisan serve
   ```

   Приложение будет доступно по адресу `http://127.0.0.1:8000`

## Функционал

### Для пользователей (посетителей)
- Регистрация с подтверждением email
- Просмотр категорий и мастер-классов
- Запись на мастер-классы (при наличии свободных мест)
- Просмотр своих записей

### Для инструкторов
- Личный кабинет
- Добавление мастер-классов
- Редактирование описания и цены мастер-классов
- Просмотр списка своих мастер-классов
- Просмотр участников каждого мастер-класса

### Особенности
- Занятые временные слоты отображаются как неактивные при добавлении мастер-класса
- Проверка доступности мест при записи (с защитой от race condition)
- Инструкторы могут входить без подтверждения email
- UI восстановлен из старого проекта (CSS и шаблоны)
- Fallback: если почта не отправляется, ссылка для подтверждения отображается на странице

## Структура проекта

- `app/Http/Controllers/` — контроллеры
- `app/Models/` — модели Eloquent
- `resources/views/` — Blade-шаблоны
- `public/Шаблоны/` — CSS и изображения из старого проекта
- `database/migrations/` — миграции БД
- `database/seeders/` — seeder для инструкторов
